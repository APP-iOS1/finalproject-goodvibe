# 국밥부장관
- 프로젝트 소개 (페르소나, ADS)
- "국밥부장관"은 지역국밥맛집처럼 숨은 맛집을 찾아다니는 사람들을 위한 앱이다.
- "국밥부장관"은 서울과 부산에 있는 동네 국밥 맛집에 대한 정보들을 제공한다. 왜냐하면 동네 국밥 맛집들은 아는 사람들만 가는 숨은 맛집이 많기 때문에 그 정보를 알기가 힘들기 때문이다.
